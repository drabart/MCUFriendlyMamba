\begin{abstract}

  Machine learning is becoming more and more widespread. While the frontier models grow in size, and their compute requirements, for Edge AI the amount of processing power and memory are usually strictly limited. This is especially true for TinyML, a subset that specializes in running machine learning on microcontrollers (MCUs).

  This trend requires new architectures, as well as adaptations of existing ones, so that they can operate within strict memory and computational limits. The Mamba architecture, which is created around State-Space Model (SSM), is particularly promising in this setting because of its compact parameterization and strong performance on long-context tasks. This work surveys recent progress in deploying Mamba models on microcontrollers and proposes architecture and optimization strategies designed specifically for MCU constraints.

  \textbf{Keywords:}
  TinyML, Mamba, Microcontroller, Edge AI

\end{abstract}
