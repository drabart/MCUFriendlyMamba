\section{Responsible Research}

This section discusses the ethical implications of the work, reproducibility considerations, and the role of AI-assisted tools in the development process.

\subsection{Ethical Considerations}

Key ethical aspects include:
\begin{itemize}
  \item Possibility of reducing power usage
  \item Enabling bad actors to run dangerous ML models on embedded devices (listening)
\end{itemize}

\subsection{Reproducibility}

The following aspects support reproducibility:
\begin{itemize}
  \item Publicly available dataset and deterministic preprocessing pipeline
  \item Ease of reproducing the model training environment
  \item Access to the microcontroller version used
  \item Consistent evaluation metrics across all experimental runs
  \item Code structured to enable rerunning of full experimental pipelines
\end{itemize}

\subsection{Use of AI Tools}

Artificial intelligence tools were used as supportive instruments during the development of this work. Their use was limited to improving code readability, assisting with debugging, and refining written text for clarity and grammatical correctness. \

Specifically, AI assistance was used for:
\begin{itemize}
  \item Debugging implementation issues in experimental code
  \item Improving clarity and structure of written explanations
  \item Assisting with formatting and consistency in LaTeX writing
\end{itemize}

All scientific decisions, experimental design choices, and interpretation of results were made independently by the author. AI tools were not used to generate or alter experimental outcomes, results, or conclusions.
