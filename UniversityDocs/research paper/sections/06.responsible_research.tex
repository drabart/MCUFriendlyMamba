\section{Responsible Research}
\label{sec:responsible_research}

This section discusses ethical considerations, reproducibility practices, and how AI tools were used during the research. It documents dataset and code licensing, replication steps, and potential misuse risks and mitigations.

\subsection{Ethical Considerations}

While this work does not involve third-party participants, it is important to consider potential impacts. The main focus is enabling more efficient machine learning on microcontrollers. Shifting some computation from datacenters to low-power devices can improve sustainability.

On-device processing can cut cloud exposure and improve privacy by keeping raw data local, but it can also be misused (for example, for covert surveillance). Deployment should therefore consider safeguards, transparency, and appropriate governance.

\subsection{Reproducibility}

The datasets used in this research are publicly available, and all software used had licenses permitting research use and has been acknowledged.

We documented the steps needed to replicate the experiments, including model training, conversion, quantization, and microcontroller benchmarking. Full replication requires the specific MCU, but the ESP32 family used here is widely available in the TinyML community.

We used a single profiler consistently across runs and annotated results with the MCU variant used to improve comparability.

The project code is publicly available and includes instructions and library versions to facilitate replication.

\subsection{Use of AI Tools}

AI tools were used as supportive instruments during development. Their use was limited to writing helper scripts, assisting with debugging, and improving clarity and grammar in written text.

Specifically, AI assistance was used for:
\begin{itemize}
  \item Writing scripts to set up the experiment pipeline
  \item Debugging implementation issues in experimental code
  \item Improving clarity and structure of written explanations
  \item Assisting with LaTeX formatting and consistency
\end{itemize}

All scientific decisions, experimental design choices, and interpretation of results were made independently by the author. AI tools were not used to generate or alter experimental outcomes, results, or conclusions.
